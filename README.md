## 工程 WebRegister 的使用方式
进入 WebRegister 目录
执行 python3 manage.py runserver 
若成功可以看到
Starting development server at http://127.0.0.1:8000/
Quit the server with CONTROL-C.

此时在浏览器中访问 http://127.0.0.1:8000/
即可看到 "主页"
